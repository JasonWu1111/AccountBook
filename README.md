# Go Wallet
Go Wallet 是一款简洁、漂亮的记账本应用。整体 UI 使用 Material Design 设计，实现了以下的功能：
- 账本的创建、分类和删除
- 账单的记入、删除、查看详情
- 自动生成每月账单的环形统计表
